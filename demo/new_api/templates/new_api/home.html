WELCOME TO GEOLOC APP



OPTIONS:


/redirect/ Redirects to ipstack
/get/ Submits your IP to DB and generates geolocation data
/api/token/ Allows for login and JWT token retrieval
/api/token/refresh/ Allows for JWT token refresh
/api/v1/geoloc/ When accessed with a token as a header(Bearer) shows the API View
/api/v1/geoloc/new/ Allows creation of geolocation instance

Hints:


/get/ can be also used with chosen IP address, like "/get/192.168.255.129


/api/geoloc/ can be also used with id of the location at the end, this allows i.e DELETE method
